Bill of Materials API Reference

    /list-all-assemblies

  • Method: GET
  • Params: None
  • Description: list all assemblies

    • /assembly/{id}

  • Method: GET
  • Params:
    • id: Part ID
  • Description: list all parts in a specific assembly (which are not sub-assemblies)

    • /combine-parts

  • Method: POST
  • Params:
    • PartId1 (Required): First part ID
    • PartId2 (Optional): Second part ID
    • AssemblyName (Required): Name to give to new assembly
  • Description: add one or more parts as "children" to a "parent" part, which then becomes an assembly
  • Example: curl -d "PartId1=1&PartId2=4&AssemblyName=NewWidget" -X POST http://<host>:<port>/combine-parts

    • /remove-part-assembly

  • Method: DELETE
  • Params:
    • PartId1 (Required): First part ID
    • PartId2 (Optional): Second part ID
    • AssemblyId (Required): Id of assembly to remove part from
  • Description: remove one or more parts from an assembly
  • Example: curl -d "PartId=1&PartId=2&AssemblyId=1" -X DELETE http://<host>:<port>/remove-part-assembly

    • /assembly-parts/{assembly_id}

  • Method: GET
  • Params:
    • assembly_id: ID of assembly
  • Description: list all the first-level children of a specific assembly

    • /get-all-assembly-parts/{assembly_id}

  • Method: GET
  • Params:
    • assembly_id: ID of assembly
  • Description: list all children of a specific assembly

    • /top-level-assemblies

  • Method: GET
  • Params: None
  • Description: list all top level assemblies (assemblies that are not children of another assembly)

    • /sub-assemblies

  • Method: GET
  • Params: None
  • Description: list all subassemblies (assemblies that are children of another assembly)

    • /component-parts

  • Method: GET
  • Params: None
  • Description: list all component parts (parts that are not subassemblies, but are included in a parent assembly)

    • /part-in-assemblies/{part_id}

  • Method: GET
  • Params:
    • part_id: ID of part
  • Description: list all assemblies that contain a specific child part, either directly or indirectly (via a subassembly)

    • /list-all-parts

  • Method: GET
  • Params: None
  • Description: list all parts

    • /part/{part_id}

  • Method: GET
  • Params:
    • part_id: ID of part
  • Description: list part by id

    • /add-part

  • Method: POST
  • Params:
    • PartName (Required): Name of part
    • PartDescription (Optional): Description of part
    • PartMaterialType (Optional): Material type
    • PartColor (Optional): Material color
  • Description: create a new part
  • Example: curl -d "PartName=MyPart&PartDescription=A%20Widget&PartMaterialType=Plastic&PartColor=Brown" -X POST http://<host>:<port>/add-part

    • /delete-part

  • Method: DELETE
  • Params:
    • PartId (Required): Id of part
  • Description: delete a part (thereby also deleting the part from its parent assemblies)
  • Example: curl -d "PartId=1" -X DELETE http://<host>:<port>/delete-part

    • /orphan-parts

  • Method: GET
  • Params: None
  • Description: list all orphan parts (parts with neither parents nor children)

    • /add-assembly-to-assembly

  • Method: POST
  • Params:
    • AssemblyId1 (Required): Child assembly
    • AssemblyId2 (Required): Parent assembly
  • Description: add child assembly to parent assembly
  • Example: curl -d "AssemblyId1=1&AssemblyId2=2" -X POST http://<host>:<port>/add-assembly-to-assembly
